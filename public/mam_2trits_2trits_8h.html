◆ trits_is_null()

bool trits_is_null ( trits_t  x)

Check x.p against null.

Note
Usually trits_t can't be null. All basic layers including trits, sponge, prng, wots, and mss rely on caller to allocate memory for trits. But in certain cases where the size of memory is difficult to trac memory can be allocated within a callee. In such case trits should be passed by pointer: trits_t *x.
