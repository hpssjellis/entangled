38 ## IOTA Entangled Monorepo
39 [![Build status](https://badge.buildkite.com/42ef0512276a270bd34cb0010fd641558b344dab4aa2225fba.svg)](https://buildkite.com/iota-foundation/entangled)
40 
41 This is a monorepo containing all you need to get [[IOTA]](https://iota.org/) operating in C/C++
42 
43 Containing IOTA components, models as well as cryptography primitives used in IOTA.

62 ## Developing Entangled
63 - Be sure to run `./tools/hooks/autohook.sh install` after initial checkout!
64 - Pass `-c dbg` for building with debug symbols.
65 
66 ### Tools required for running git commit hook
67 - buildifier
68 - clang-format
69 
70 ### Buildifier
71 Buildifier can be installed with `bazel` or `go`
72 
73 #### Install with go
74 1. change directory to `$GOPATH`
75 2. run `$ go get github.com/bazelbuild/buildtools/buildifier`
76  The executable file will be located under `$GOPATH/bin`
77 3. make a soft link for global usage, run
78  `$ sudo ln -s $HOME/go/bin/buildifier /usr/bin/buildifier`
79 
80 #### Install with bazel
81 1. clone `bazelbuild/buildtools` repository
82  `$ git clone https://github.com/bazelbuild/buildtools.git`
83 2. change directory to `buildtools`
84 3. build it with bazel command, `$ bazel build //buildifier`
85  The executable file will be located under `path/to/buildtools/bazel-bin`
86 4. make a soft link
87 
88 ### clang-format
89 clang-format can be installed by command:
90 - Debian/Ubuntu based: `$ sudo apt-get install clang-format`
91 - OSX: `$ brew install clang-format`
